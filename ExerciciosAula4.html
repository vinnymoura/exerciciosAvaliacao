QUESTIONÁRIOS

1- O que você entendeu por algoritmos ?

Algoritmo é uma sequência de instruções para resolver problema e/ou executar uma ação.

2- Explique com suas palavras o que é uma linguagem de programação ?

Conjunto de instruções e/ou regras que o ser humano encontrou de ordenar um computador a executar ações, que compiladas se tornam um programa.

3- Qual a diferença de linguagens compiladas/interpretadas ?

A linguagem interpretada é “traduzida” por um terceiro e a linguagem compilada traduz o programa diretamente.

4- Quais os tipos de dados que existem ?

Numéricos, caracteres e lógicos

5- O que é atribuição ?

Atribuição é dar um valor a uma variável ou constante.

6- Qual a diferença entre constante e variável ?

Constante não pode ser alterado no decorrer da execução do programa enquanto, variáveis podem ser alteradas quantas vezes forem necessárias
